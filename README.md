##youtu
`youtu`是http://open.youtu.qq.com 提供接口的Go实现版本


### 下载:

```bash
go get https://github.com/Tencent-YouTu/go_sdk
```

### 使用:
1. 到http://open.youtu.qq.com 注册成为开发者
2. 注册应用，获取开发者密钥
